# Полурывок 2026 — платформа онлайн-регистрации

Сайт регистрации на открытый онлайн-турнир в рывке старым стилем (полурывке).
Хостинг — **GitHub Pages** (бесплатно, без покупки домена). Заявки сохраняются
в **Google Таблицу** через Google Apps Script — она же служит журналом учёта
участников для оргкомитета.

## Что внутри

| Файл | Назначение |
|---|---|
| `index.html` | Выбор турнира, регистрация, оплата взноса (QR), список участников |
| `privacy.html` | Политика конфиденциальности (152-ФЗ) |
| `apps-script/Code.gs` | Бэкенд: приём заявок в таблицу, письмо-подтверждение, публичный список |

## Два турнира

Заявка общая, участник отмечает один или оба турнира:

| Турнир | Взнос | QR-код |
|---|---|---|
| Ко Всероссийскому дню физкультурника | 2000 ₽ | `qr/qr-fizkulturnik.png` |
| К 141-й годовщине гиревого спорта | 4000 ₽ | `qr/qr-141.png` |

Оплата — на карту Сбербанка 2202 2061 5850 1465, у каждого турнира **свой QR**.
Положите картинки QR-кодов от банка в папку `qr/` с именами из таблицы — они
появятся в блоке оплаты автоматически. Пока файлов нет, показываются только
реквизиты карты. Чек участник отправляет в Max +7 914 567-94-82 или на
rsl32@mail.ru. В таблице учёта есть столбец «Взнос оплачен» — отметку ставит
организатор вручную после проверки чека.

## Что умеет форма

- ФИО, дата рождения, пол, страна/город, клуб, email, телефон
- **Группа (1–9)** подставляется автоматически по возрасту на дату турнира
  (08.08.2026); заявиться в более младшую группу можно, в более старшую — нет
- **Вес гирь** выбирается «клеймами» 4–36 кг; клейма ниже минимума для группы
  и пола автоматически блокируются
- **Весовая категория** считается сама: открытая группа — до/свыше 80 кг (муж.)
  и до/свыше 64 кг (жен.); ветераны — без категорий
- Выбор турнира(ов) с расчётом суммы взноса и показом нужных QR-кодов
- Почтовый адрес для отправки наград призёрам
- Два обязательных согласия (обработка ПДн + ответственность за здоровье)
  и отдельное необязательное согласие на публикацию в открытом списке
- Защита от ботов (honeypot) и от повторной заявки с одного email
- Участнику автоматически уходит письмо-подтверждение

## Развёртывание — 2 шага

### Шаг 1. Google Таблица + Apps Script (приём заявок)

1. Создайте новую Google Таблицу (например, «Полурывок 2026 — участники»)
   в аккаунте sg@fsegas.com.
2. В таблице: **Расширения → Apps Script**. Удалите содержимое и вставьте код
   из `apps-script/Code.gs`. Сохраните (Ctrl+S).
3. При желании в начале файла укажите `ORGANIZER_EMAIL` — на него будут
   приходить уведомления о каждой новой заявке.
4. **Развернуть → Новое развёртывание → тип «Веб-приложение»**:
   - *Выполнять от имени:* **Меня**
   - *У кого есть доступ:* **Все** (иначе форма не сможет отправлять заявки)
5. Нажмите «Развернуть», разрешите доступ (Google предупредит — это ваш
   собственный скрипт, жмите «Дополнительно → Перейти»).
6. Скопируйте **URL веб-приложения** (`https://script.google.com/macros/s/.../exec`).
7. Откройте `index.html` и вставьте этот URL в самом начале блока `<script>`:

   ```js
   const APPS_SCRIPT_URL = "https://script.google.com/macros/s/…/exec";
   ```

Лист «Участники» с шапкой создастся автоматически при первой заявке.

### Шаг 2. GitHub Pages (сайт)

1. Создайте публичный репозиторий, например `polusnatch-2026`.
2. Загрузите в корень `index.html`, `privacy.html` (папку `apps-script`
   можно тоже — она не мешает).
3. **Settings → Pages → Source: Deploy from a branch → main / (root) → Save.**
4. Через 1–2 минуты сайт доступен по адресу:
   `https://<ваш-логин>.github.io/polusnatch-2026/`

Эту ссылку и указывайте в положении, соцсетях и Telegram-канале.

## Перед публикацией — обязательно

- [ ] В `privacy.html` в разделе 1 укажите **оператора персональных данных**
  (юр. лицо или ФИО организатора) — там оставлен комментарий-заглушка.
- [ ] Проверьте контактный email в `index.html` и `privacy.html`
  (сейчас указан sg@fsegas.com).
- [ ] Отправьте тестовую заявку и убедитесь, что она появилась в таблице
  и пришло письмо-подтверждение.

## Защита данных и доступ администратора

Как устроена безопасность:
- Полная база лежит в приватной Google Таблице владельца — с сайта её не видно.
- Открытый список на сайте отдаёт только ФИО, город, группу, гири и турнир —
  и только для тех, кто поставил галочку согласия на публикацию. Почта,
  телефон, дата рождения, вес и адрес наружу не выходят никогда.
- Полную базу можно скачать только по секретному ключу администратора.

**Настройка ключа (1 минута).** В редакторе Apps Script в начале `Code.gs`
замените заглушку `ADMIN_KEY` на длинный случайный ключ (20+ символов, буквы
и цифры) и переразверните веб-приложение. ⚠️ Реальный ключ живёт только в
редакторе Apps Script — в GitHub коммитьте файл с заглушкой.

**Ссылка для тренера.** Отправьте ему в личку ссылку вида:

    https://script.google.com/macros/s/…/exec?key=ВАШ_КЛЮЧ

Открыл ссылку — скачался CSV со всей базой (открывается в Excel и Numbers,
кириллица не ломается). Работает с телефона, Google-аккаунт не нужен.
Если ключ утёк — просто поменяйте его в скрипте и пришлите тренеру новую ссылку.

Дополнительно можно расшарить саму Google Таблицу на аккаунт тренера
(режим «Читатель») — тогда он видит данные вживую.

## Логотипы организаторов

Положите PNG-файлы в папку `logos/` с именами `logo_100pudov.png`,
`logo_rudnev.png`, `logo_rlgm.png`, `logo_pud.png`, `logo_fse.png`
(те же, что в проекте PDF-положения). Появятся в секции «Организаторы»
автоматически; пока файла нет — карточка показывает название текстом.

## Учёт участников (для оргкомитета)

Вся работа — в Google Таблице: сортировка по группам, фильтры по весу гирь,
отметки о полученных видео (добавьте свои столбцы правее — скрипт их не
трогает). Стартовые протоколы удобно собирать сводной таблицей по столбцам
«Группа» и «Весовая категория».

## Изменение данных / отзыв согласия

Участник пишет на контактный email — организатор правит или удаляет строку
в таблице вручную. После удаления строки участник исчезает и из открытого
списка на сайте.

## Ограничения

- Apps Script (бесплатный тариф Workspace): до ~1500 писем в день и десятки
  тысяч запросов — для турнира этого более чем достаточно.
- Список участников на сайте обновляется при каждой загрузке страницы.
